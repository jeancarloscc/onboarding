# Repositório de Onboarding - Laboratório de Inteligência de Dados

Bem-vindo ao repositório de Onboarding!

Este repositório reúne os manuais, materiais de apoio e atividades de ambientação para novos membros do Laboratório de Inteligência de Dados. O objetivo é facilitar sua integração e garantir que você tenha o suporte necessário para iniciar sua jornada conosco.

---

## Conteúdo

### 👥 Manual do Aluno
Documentação voltada para os alunos e membros da equipe:
- Ambientação no Git e GitHub
- Fluxo de trabalho e colaboração
- Padrões de versionamento (**GitFlow**)
- Boas práticas de desenvolvimento

Acesse: [`/alunos`](./alunos)

---

### 🎓 Manual do Líder
Documentação voltada para quem atua como líder:
- Responsabilidades e competências do líder
- Apoio técnico e mentoria nas atividades
- Boas práticas de liderança e gestão técnica
- Acompanhamento e desenvolvimento do time

Acesse: [`/lider`](./lider)

---

## Começando

1. Acesse a pasta correspondente ao seu papel (Aluno ou Líder).
2. Leia o material disponível com atenção.
3. Realize as atividades práticas propostas.
4. Em caso de dúvidas, procure seu líder ou supervisor direto.
