# 👥 Manual do Aluno - Laboratório de Inteligência de Dados

Este manual tem como objetivo orientar alunos e membros da equipe sobre o fluxo de trabalho que deve ser seguido durante a execução das atividades no contexto dos projetos do Laboratório de Inteligência de Dados. O documento descreve as boas práticas, ferramentas e responsabilidades no dia a dia.

## O que é o Manual do Aluno?

Este manual reúne orientações para apoiar sua atuação no time, com foco em colaboração, organização das atividades e boas práticas de versionamento de código.
